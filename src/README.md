# Cartilla Digital - Resguardo Indígena La Gaitana

**YU' WE'SX - SABERES DE NUESTROS MAYORES**

Cartilla digital interactiva del Resguardo Indígena La Gaitana, desarrollada con Angular 17 y Tailwind CSS.

## Descripción

Esta aplicación web presenta los conocimientos ancestrales del pueblo Nasa/Guanaco del Resguardo Indígena La Gaitana, ubicado en Inzá, Cauca, Colombia.

## Tecnologías Utilizadas

- **Angular 17** (Standalone Components)
- **Tailwind CSS 4.0** (para estilos)
- **TypeScript** (lenguaje de programación)
- **Web Speech API** (para reproducción de audio de pronunciaciones)

## Instalación

1. Instalar las dependencias:
```bash
npm install
```

2. Iniciar el servidor de desarrollo:
```bash
npm start
```

3. Abrir el navegador en `http://localhost:4200`

- ✅ Navegación fluida entre 10 secciones
- ✅ Diseño responsive (móvil y escritorio)

- ✅ Componentes standalone de Angular

- ✅ Interfaz cultural apropiada con colores tradicionales

## Secciones de la Cartilla

1. **Introducción** - Identidad NASA del resguardo
2. **Origen del Pueblo Nasa** - Historia y cosmogonía
3. **Lengua Nasa Yuwe** - Vocabulario básico con audio
4. **Medicina - Diarrea** - Remedios tradicionales
5. **Medicina - Fiebre** - Plantas medicinales
6. **Uso Animal** - Medicina tradicional con animales
7. **Ritualidad** - Ceremonias y cosmovisión
8. **Historias** - Relatos de origen y lucha territorial
9. **Cronología** - Línea de tiempo histórica
10. **Agradecimientos** - Reconocimientos y referencias
